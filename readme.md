# @rmw/env

##  安装

```
npm install @rmw/env
```

## 使用

```
import DIR_CACHE from '@rmw/env/dir/cache'
import DIR_CONFIG from '@rmw/env/dir/config'

console.log DIR_CACHE, DIR_CONFIG
```
